# Модуль AI Agent Actions для OkayCMS

Дозволяє користувачам швидко аналізувати товари за допомогою різних AI-сервісів прямо зі сторінки товару.  
Модуль додає кнопки для швидкого доступу до ChatGPT, Gemini, Grok, Perplexity та Claude з автоматично сформованим промптом на основі інформації про товар.

## Можливості

- Швидкий аналіз товару за допомогою ChatGPT
- Аналіз товару за допомогою Google Gemini
- Аналіз товару за допомогою Grok
- Аналіз товару за допомогою Perplexity
- Аналіз товару за допомогою Claude
- Автоматичне формування промпту з інформацією про товар (назва, характеристики, опис, URL)
- Автоматичне копіювання промпту в буфер обміну
- Відкриття AI-сервісу у новій вкладці з готовим запитом

## Вимоги

- OkayCMS v4.5.2

## Встановлення

### Варіант 1: Скачування з GitHub

1. Скачайте останню версію модуля з [останнього релізу на GitHub](https://github.com/devSviat/AIAgentActions-OkayCMS/releases/latest).  
2. Розпакуйте архів на сервері в каталог: `{OkayCMS_root}/Okay/Modules/`

### Варіант 2: Встановлення через Git

1. Виконайте команду в директорії `{OkayCMS_root}/Okay/Modules/Sviat`:
   ```bash
   git clone https://github.com/devSviat/AIAgentActions-OkayCMS.git AIAgentActions
   ```

## Використання

### Фронтенд

Модуль автоматично додає блок з кнопками AI-сервісів на сторінку товару після розділу характеристик. При натисканні на кнопку:

1. Модуль збирає інформацію про товар (назва, URL, характеристики, опис)
2. Формує структурований промпт для аналізу товару
3. Відкриває вибраний AI-сервіс у новій вкладці з готовим запитом
4. Копіює промпт в буфер обміну для подальшого використання

### Адмін-панель

Модуль додає розділ "AI Agent Actions" в адмін-панелі з описом функціональності та інструкціями по використанню.

## Як це працює

При натисканні на кнопку AI-сервісу модуль формує детальний промпт, який включає:

- Назву товару
- URL сторінки товару
- Характеристики товару (якщо доступні)
- Опис товару (обрізаний до 500 символів, якщо довший)

Промпт містить структурований запит до AI з проханням:
1. Дати короткий висновок про товар
2. Пояснити ключові характеристики
3. Вказати основні переваги
4. Вказати можливі недоліки або обмеження
5. Визначити цільову аудиторію
6. Порівняти з аналогами (якщо можливо)
